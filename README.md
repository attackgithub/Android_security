## 代码审计

* 安卓 app
	历史漏洞
    掌握漏洞所在模块或子系统，不看完整的漏洞细节描述，尝试在漏洞版本中找出对应的漏洞
    如果未能找出漏洞，去看漏洞细节描述，对比自己审计过程，看遗漏了哪一步骤
    直到相信：挖洞只是体力消耗，而非能力问题

* 项目所涉及安卓 app
